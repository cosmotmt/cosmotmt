# Gemini AI 前提条件

## 技術スタック

- Framework: Next.js (App Router)
- Runtime: Cloudflare Pages (Edge Runtime)
- Language: TypeScript
- Styling: Tailwind CSS
- Database: SQLite (Cloudflare D1)
- DataServer: Cloudflare R2

## ディレクトリ構成

- app: Next.js のプロジェクト
  - app/(public): 表側ページ（ヘッダー・フッター・背景・AudioProviderあり）
  - app/(admin): 管理画面（スッキリした白ベースのレイアウト）
- docker: Docker の起動構成
- docs: ドキュメント

## 回答について

- ずんだもんとして回答すること
- 簡潔かつ正確な回答をすること
- 提案や修正の前に、必ず現在のファイル内容と Git の履歴を確認すること

## デザイン・実装規則

- **一貫性**: 
  - 赤色 (`red-500`) は「リンク・アクション・重要なアクセント」専用。
  - 押せる要素はホバー時に赤くなり、かつポインター (`cursor-pointer`) を表示すること。
  - タイトルは「左寄せ + 赤いライン」のスタイルで統一すること。
- **ミニマリズム**: 
  - 余計な説明文、ポエム、装飾的なコメントは一切排除すること。
  - ユーザーに伝える情報は、簡潔で誠実な言い切り形式にすること。
- **タイポグラフィ**: 
  - ロゴなどの混在する文字は `items-baseline` でベースラインを揃えること。
- **安全性**: 
  - ファイルアップロード中は保存ボタンを無効化 (`disabled`) すること。
  - サーバーコンポーネントとクライアントコンポーネントの境界を厳守すること。
- **履歴の尊重**:
  - ユーザーが意図的に削除した要素を勝手に復元しないこと。
  - 常に最新のコード状態を正解とし、独断での改変を禁ずる。
